Das ist die LuckySite Artikelfunktion:

Mit der LuckySite Artikelfunktion ist es einfach, gutaussehende und responsive Artikel auf der Website zu veröffentlichen.

Wichtige Klassen:

Für Text:

Mit center kann der Text zentriert werden.

Mit code können Textteile wie Code dargestellt werden. Die Klasse sollte in inline <span> Elementen eingesetzt werden.

Für Bilder:

float_img_left setzt die maximale Bildbreite auf 50% und und zentriert es links. Außerdem Umfließt der nachfolgende Textabschnitt (das nachfolgende Element) das Bild. Das Bild wird also in den Text integriert.

float_img_right setzt die maximale Bildbreite auf 50% und und zentriert es rechts. Außerdem Umfließt der nachfolgende Textabschnitt (das nachfolgende Element) das Bild. Das Bild wird also in den Text integriert.

Hinweise zum Aufbau:

Die Content-Section

Das übergeordnete <section> Element muss die Klasse article_container erhalten.

Der Inhalt

Der Inhalt muss in ein <article> Element eingefügt werden.

Dabei kann der Inhalt aus ganz normalen Elementen wie z.B. <h1> <h2> <p> <img>bestehen.

Soll der Artikel in mehrere Sektionen eingeteilt werden, muss das <article> Element die Klasse article_multiSec erhalten. Dann kann der Inhalt in verschiedene <section> Elemente eingefügt werden.
